== Description ==

The Accordion block allows you to create collapsible content sections within your WordPress posts and pages. 
This is particularly useful for FAQs, where you want to present a list of questions that can be expanded to reveal the answers.

Each section can be expanded or collapsed individually, providing a clean and organized way to display content.
The block is fully customizable, allowing you to adjust the appearance to match your site's design.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/accordion` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= How do I add an accordion block to my post or page? =

To add an accordion block, simply click on the "+" button to add a new block, search for "Accordion," and select it. You can then add your content to the accordion sections.

= Can I customize the appearance of the accordion block? =

Yes, you can customize the appearance of the accordion block using the block settings in the editor. You can change colors, and other styles to match your site's design.

= Is the accordion block responsive? =

Yes, the accordion block is fully responsive and will adjust to fit different screen sizes, ensuring a good user experience on both desktop and mobile devices.

= How do I collapse all sections by default? =

In the block settings, you can choose to have all sections collapsed by default. This setting can be found under the "Behavior" options.

= Can I use multiple accordion blocks on a single page? =

Yes, you can add multiple accordion blocks to a single post or page. Each block will function independently, allowing you to organize your content as needed.
